New HTML Project

最小の素のHTML/JSだけのプロジェクト

使い方

  1. このページを編集して保存（このファイル: projects/simple_html/index.html）
  2. ローカルで表示: make web-simple-html
  3. Gitへ反映: Workbench か Dev Agent を使う（どちらか）
    • Workbench: make web-workbench → Enterで add/commit/push
    • Dev Agent（ローカルAPI）: make web-dev-agent
    • Remote（APIのみ）: make open-dev-remote（owner/repo/path を入力）
リポジトリ直下の origin は https://github.com/attrip/xxx を指しています。
